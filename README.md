# jQuery Cool Grid

calclate item's height and show items

## Install

```
npm install -g coffee-script

coffee -c jquery.coolgrid.coffee
```

## Usage

```js
options = {
  autoResize: true,
  container: $('.items'),
  offset: 5,
  item_width: 210
}

$('.item').CoolGrid(options);
```
